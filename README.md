# 기획부터 시스템 연동까지, 화면 너머를 책임지는 개발자
기획부터 디자인, 프론트엔드와 연동까지, 사용자에게 실질적인 가치를 제공하는 서비스를 만드는 데 관심이 많은 개발자입니다.
프론트엔드 개발자로서 아키텍처 설계와 UI 구현은 물론, 백엔드 및 실시간 통신(WebSocket) 연동까지 전체 개발 사이클을 주도하여 안정적이고 사용성 높은 서비스를 만들어냅니다.

##  About Me

* **관심 분야:** 모바일/웹 어플리케이션 개발, 자율주행 서비스 연동, UI/UX 디자인
* **개발자로서의 목표:** 단순한 화면 구현을 넘어, 백엔드 및 다양한 인프라와의 원활한 데이터 통신을 통해 안정적이고 사용성 높은 사용자 중심 서비스를 설계하는 개발자
* **성장 방식:** 요구사항에 맞춘 기획을 바탕으로, 다양한 기술 스택을 상황에 맞게 도입하고 응용하며 문제를 해결해 나갑니다.

* **개발 기간:** 진행 중 (~ 2026.03.31)

* **주요 기능 및 구현 내용:**
  * 로보택시 호출, 출발지-도착지 설정 및 경로 미리보기 기능 구현
  * 예상 소요 시간 및 요금 확인 및 자동 결제 인터페이스 구축

  * 사용자 로그인/로그아웃 및 고객센터(문의사항) 기능 구현

* **Kotlin:** 안드로이드 어플리케이션 개발을 위한 주력 언어로 활용하며, 단독(1인) 프론트엔드 개발자로서 프로젝트의 클라이언트 아키텍처 설계부터 UI 구현까지 전체 개발 사이클을 주도한 경험이 있습니다. 기능 중심(Feature-driven) 패키지 구조와 MVVM 패턴을 적용하여 확장성 있고 유지보수하기 좋은 코드를 작성할 수 있습니다. 특히, 웹소켓(WebSocket) 통신을 통해 자율주행 시뮬레이터(ROS) 시스템과 실시간으로 연동하여, 차량의 주행 데이터 및 경로를 지도 상에 즉각적으로 시각화하는 등 복잡한 외부 시스템과의 데이터 연동을 안정적으로 구현할 수 있습니다.
* **Coroutine:** 비동기 프로그래밍을 위해 코루틴을 적절하게 활용할 수 있습니다. 서버 API 통신이나 웹소켓 연결 등 시간이 소요되는 작업을 메인 스레드 차단 없이 백그라운드에서 처리하여, 사용자가 앱을 사용할 때 화면이 멈추지 않고 원활하게 동작하도록 구현할 수 있습니다.
* **Jetpack Compose:** 선언형 UI 툴킷인 컴포즈를 활용하여 안드로이드 화면을 구성할 수 있습니다. 데이터의 상태(State) 변화에 따라 화면이 자동으로 업데이트되는 원리를 이해하고 있으며, 복잡한 레이아웃과 디자인을 직관적이고 재사용 가능한 코드로 구현할 수 있습니다.
* **Git:** 브랜치 전략을 활용하여 팀원들과 원활하게 코드를 병합(Merge)하며 프로젝트를 진행할 수 있습니다. 이 과정에서 발생하는 다양한 코드 충돌(Conflict) 상황을 이해하고 적절하게 해결한 경험이 있습니다.
* **Jira:** 애자일(Agile) 기반의 협업 환경에서 일주일에 30~40 정도의 스토리 포인트를 산정하여 체계적으로 스프린트를 관리했습니다. 특히 정확한 번다운 차트(Burndown Chart) 기록을 위해 4가지 자체 규칙(1. 스프린트 시작 전 백로그 계획 완료, 2. 작업을 1~3일 단위의 작은 포인트로 세분화, 3. 매일 '완료(Done)' 상태 최신화, 4. 스프린트 도중 새 작업 추가 지양 및 필요시 작업 교체)을 수립하고 준수하여 성공적으로 프로젝트 일정을 관리했습니다.
* **Django:** Django Framework를 이해하고 명세에 맞춰 웹 서버를 구현할 수 있습니다. 데이터 스키마를 사용하여 적절한 Model을 설정하고 마이그레이션 할 수 있으며, ORM을 활용하여 CRUD를 구현합니다. Static 파일 처리와 인증/권한(로그인, 로그아웃) 시스템을 서버에 구성할 수 있습니다.
* **HTML/CSS:** 시맨틱 마크업을 이해하고 Block/Inline 요소를 구조화할 수 있습니다. CSS Box 모델, Position, Display를 활용해 레이아웃을 적절히 배치하며 CDN을 적용해 성능을 최적화할 수 있습니다.
* **Bootstrap:** 컴포넌트와 Grid/Flexbox 시스템을 활용하여 모바일/PC 반응형 웹 페이지를 빠르게 구현하며, JS 없이도 Modal, Dropdown 등의 인터랙티브 UI를 구성할 수 있습니다.
* **SQLite:** 관계형 데이터베이스의 테이블, 컬럼 구조를 이해하고 정규화할 수 있습니다. Primary/Foreign Key를 활용한 모델링과 조건문을 활용한 데이터 CRUD 작업, 애플리케이션 연동이 가능합니다.
* **JavaScript:** DOM 조작을 통해 HTML 요소를 동적으로 제어하고, 브라우저 이벤트 핸들링을 구현할 수 있습니다. Ajax와 Axios를 활용해 API 서버와 비동기 HTTP 통신을 수행합니다.
* **Vue.js:** 조건 및 반복 렌더링을 통한 동적 UI 구현과 이벤트 처리가 가능하며, Vue Router를 활용해 SPA(Single Page Application) 환경의 라우팅을 구현할 수 있습니다.
* **PyTorch:** 파라미터 업데이트와 손실함수 최소화 과정을 이해하고, GPU 환경 설정 및 텐서 연산 최적화를 통해 연산을 수행할 수 있습니다.
* **Hugging Face:** 특정 도메인 요구사항에 맞추어 모델의 성능을 향상시키는 파인튜닝 작업을 수행할 수 있습니다.
* **Pandas / Numpy:** 데이터 통계(mean, std, sum) 분석, 이상치 탐지, 전처리 및 표준화 작업을 수행하며 행렬 곱을 비롯한 머신러닝 필수 수치 연산을 다룰 수 있습니다.
